Java中的那些属性和方法

java类的成员变量包括实例变量和类变量，成员方法包括实例方法和类方法。

成员变量

实例变量

，是新建了实例后才能用的变量

类变量，

是不用新建实例，直接通过类来调用的变量。特点是用static修饰。所以它们还有个响当当的名字，叫静态变量。
成员变量最大，包括实例变量和类变量，类变量=静态变量

成员方法

实例方法

类方法

注意 在不是继承关系的情况下，每new 一次对象 就生成一个实例变量

总得来说，Java中的属性是绑定到类型上，方法是绑定到对象上
